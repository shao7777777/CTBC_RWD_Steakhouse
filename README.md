# UI/UX 響應式深度實踐：高質感餐飲服務網站
專案說明 (Project Description)
--
此專案為一個高質感的牛排館主題網站，旨在將 **UI/UX 思考導入 RWD 實踐**。專案的核心挑戰在於：如何透過介面設計與排版，在電腦網頁和行動裝置上有效地傳達品牌故事與情感連結，並處理複雜的**餐點菜單**與**互動式內容**，以確保在行動裝置上的易用性。排版上輔以 **Bootstrap 框架，提升開發效率**。

核心技術與設計理念
--
1. 技術棧 (Tech Stack)
 - 語言： HTML5, CSS3, 少量 JavaScript
 - 框架/佈局： Bootstrap 5 (元件與排版輔助), CSS Grid 與 Flexbox 混合佈局

2. 設計理念 (Design Philosophy)
 - **品牌故事與情感連結**： 介面設計旨在傳達牛排館對頂級食材的用心與品牌故事。透過精心設計的版面與配色，為顧客帶來視覺與心靈的雙重滿足，強化品牌的獨特性。
 - **高質感配色與互動**： 主題色採用沉穩的紅酒色與溫暖的米色作為基調，營造高雅氛圍。互動元素（如按鈕、連結）的 hover 效果加入暗金色點綴，藉此細節提升介面的整體質感與使用者體驗。
 - **觸控友善與結構精準** (Grid/Flexbox)： 確保導覽列、按鈕和互動元件的觸控目標尺寸符合移動裝置的最佳實踐。所有核心排版 (特別是菜單與圖片佈局) 均嚴格利用 CSS Grid 與 Flexbox 進行規劃，確保網頁在電腦網頁及移動用戶上都能提供合理、舒適的設計體驗與佈局穩定性。
